# Rebuild template download using a reliable method

**What's changing:**

The "Download Template CSV" buttons on both the Import Fields and Import Inventory screens will be completely rewritten to use a different, more reliable download method.

**How it will work:**
- On iPhone/Android: The template file is saved to a temporary location, then the system share sheet opens so you can save it to Files, email it, AirDrop it, etc.
- On web: The file downloads directly to your computer's Downloads folder.
- A success message confirms when the template is ready.

**What stays the same:**
- The CSV file format and content (field names, sample rows) remain identical.
- The button appearance and placement stay the same.
- Everything else on the Import screens is untouched.
